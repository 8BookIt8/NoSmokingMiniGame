# NoSmokingMiniGame

[![Youtube](https://img.shields.io/badge/youtube-북잇-red?logo=youtube)](https://www.youtube.com/channel/UCllpUa_1bga-GTtgL3A-40Q)
### 흡연 예방을 위한 미니게임 플러그인입니다.
### MiniGame plugin to prevent smoking.

## Usages
* /nsmg \<start> \<playerName>
  * 게임을 시작합니다
  * Start the game.
####
* /nsmg \<stop>
    * 게임을 중지합니다
    * Stop the game.
